## Getting Started

  

To get a local copy up and running follow these simple steps.

  

### Prerequisites
  
* nodejs, npm
* docker
* docker-compose

### Installation

1. Clone the repo

```sh
git clone https:://github.com/kamhix/react-tic-tac-toe.git
```

2. Install NPM packages and Launch
#### App
```sh
cd app
npm install
npm start
```
#### Api
```sh
cd server
npm install
npm start
```
or
```sh
cd server
docker-compose up
```


2. Run the test
#### App
```sh
cd app
npm install
npm test
```
#### Api
```sh
cd server
npm install
npm test
```
